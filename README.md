## 🇬🇧 English

### ✨ Features

* 🧠 **Smart Detection**

  * If Node.js is already installed → skips installation
  * If not → opens installer automatically

* 📂 **GUI Folder Selection**

  * Choose install location via Browse window (no manual typing)

* 💾 **Custom Install Paths**

  * Install Node.js anywhere (D, E, external drives…)
  * Install Qwen in a separate custom folder

* ⚡ **No C Drive Pollution**

  * No forced installation in `C:\Program Files`

* 🔧 **Automatic Setup**

  * Configures npm global directory
  * Adds Qwen to system PATH

---

### ▶️ Usage

1. Download the `.bat` file
2. Run it (double click or via CMD):
```bash
install-qwen.bat
```
3. ```bash curl -L -o "%TEMP%\install-qwen.bat" https://raw.githubusercontent.com/MoGLCL/Qween-Code-Installer-/refs/heads/main/install-qwen.bat && start "" "%TEMP%\install-qwen.bat"
   ```

```bash
install-qwen.bat
```

---

### 📁 Example

If you choose:

```
D:\Apps
```

Result:

```
D:\Apps\nodejs
D:\Apps\qwen-global
```

---

### ⚠️ Notes

* Make sure to install Node.js in the same path you selected
* You may need to restart CMD after installation
* Recommended to run as Administrator

## 🇸🇦 العربية

### ✨ المميزات

* 🧠 **تحديد ذكي**

  * لو Node.js متثبت → السكربت يكمل علطول
  * لو مش موجود → يفتحلك التثبيت

* 📂 **اختيار المكان بنافذة**

  * تقدر تختار مكان التثبيت من Browse

* 💾 **تخصيص المسارات**

  * تثبيت Node.js في أي درايف
  * تثبيت Qwen في فولدر منفصل

* ⚡ **بدون استخدام إجباري لـ C**

  * مش هيتثبت في `C:\Program Files` غصب عنك

* 🔧 **إعداد تلقائي**

  * بيظبط npm
  * بيضيف Qwen للـ PATH
